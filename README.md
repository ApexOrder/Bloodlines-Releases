# ApexOrder Bloodlines Releases

This public repository is used only for ApexOrder Bloodlines release distribution.

Source code and development files live in private repositories. Players and the launcher only use the GitHub Releases published here.

## Release tag format

- `launcher-vX.X.X` — launcher installer, portable executable, and updater ZIP
- `bloodlines-vX.X.X` — Bloodlines mod package and manifest

## Player download

Players should download the latest `ApexOrder-Bloodlines-Launcher-Setup.exe` from the newest `launcher-v*` release.

The launcher then manages Bloodlines installation, repair, and future updates automatically.
